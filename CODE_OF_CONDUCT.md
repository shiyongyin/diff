# Code of Conduct

为了让 `Tenant Diff` 成为一个专业、友好、可持续协作的项目，我们要求所有参与者遵守以下行为准则。

## 我们鼓励的行为

- 尊重不同背景、经验和观点的贡献者
- 基于事实、证据和技术讨论问题
- 对新手保持耐心，愿意解释上下文与约束
- 在评审中关注代码和方案本身，而非针对个人
- 对错误保持透明，并积极修正

## 不可接受的行为

- 人身攻击、羞辱、威胁、歧视或骚扰
- 恶意引战、反复刷屏、故意制造协作障碍
- 未经允许公开他人的敏感信息
- 明知会破坏协作氛围仍持续使用冒犯性表达

## 执行原则

维护者有权对违反本准则的行为采取适当措施，包括但不限于：

- 提醒或警告
- 暂时冻结讨论
- 拒绝相关贡献
- 在严重情况下限制参与资格

## 报告方式

若你遇到严重不当行为，请通过维护者的私有沟通渠道报告，而不要在公开讨论中继续升级冲突。

维护者会尽可能基于事实进行处理，并尊重报告人的隐私。
