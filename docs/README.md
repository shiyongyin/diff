# 文档索引

本目录包含 Tenant Diff 项目的三份 SSOT（单一权威源）文档。

## SSOT 文档

| 文档 | 定位 | 目标读者 |
|------|------|---------|
| [产品需求文档 (PRD)](prd.md) | 产品概述、快速体验、功能规格、API 合约、业务规则、版本策略 | 产品经理、前端开发、实施顾问 |
| [开发设计文档](design-doc.md) | 架构、领域模型、核心流程、SPI 扩展点、插件开发指南、数据库设计 | 后端开发、架构师 |
| [运维手册](ops-guide.md) | 部署、配置、监控、故障排查、发布管理、安全运维、CI/CD | 运维工程师、SRE |

## 推荐阅读顺序

1. **首次接触**：先看 `prd.md` 的"快速体验"章节
2. **开发扩展**：看 `design-doc.md` 的"插件开发指南"章节
3. **生产部署**：看 `ops-guide.md` 的"部署指南"和"故障排查"章节
